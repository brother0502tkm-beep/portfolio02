# アクセサリーショップ LP

## ■ 公開URL
https://brother0502tkm-beep.github.io/portfolio02/

## ■ リポジトリ
https://github.com/brother0502tkm-beep/portfolio02

## ■ 概要
アクセサリーショップをテーマにしたランディングページです。
オンラインスクール課題として制作し、Figma デザインカンプの再現性を目的にしています。

レスポンシブ対応や Sass を用いた効率的なコーディングの練習として作成しました。

## ■ 制作の目的・背景
- デザインカンプの再現性を高めるため
- スマホ・タブレットへのレスポンシブ対応の習得
- UI を意識したアニメーション実装の練習

## ■ 使用技術
- HTML
- CSS / Sass
- JavaScript
- jQuery

## ■ こだわったポイント
- ファーストビューのアニメーション演出
- ハンバーガーメニューの開閉アニメーション
- スムーススクロールを用いた操作性向上
- 各セクションの余白・視認性を意識したデザイン再現
- Sassを使った効率的なスタイル管理（変数・ネスト・mixin 等）

## ■ 制作期間
2週間

## ■ 担当範囲
Figma デザインカンプをもとにコーディング全般を担当
